Registered Users

Registered users have signed in or authenticated with your {{ site.product }} app. You can use this module to help answer user questions and resolve issues.

Search users
  1. In Find users, start typing either the email address or the name of the user you want to search for. You should see a list of users that match.
  2. Select a user from the search results list.
  3. The details page will open for the selected user.
  4. When you return to the list page, the viewed user will be added to the list.
Users list
  • E-mail - Email address for the user
    • To view user details and update, click on an E-mail link.
    • Click the Back to Users panel to return.
  • Name - The display name for the user
Note: You will only see the users that you have searched for and selected.
