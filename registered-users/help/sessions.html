Active Sessions

Here are the active sessions associated with a registered user.

Session details

  • Date - Date when a session was started; click the header to control the sort order
  • Device - Type of device being used for a session
  • Start time - Time when a session was started
  • End Session - Click associated with a session to delete it; this will free up a license, and allow you to login to an available session
  • End all sessions - Click End all sessions to terminate all active sessions for this user
Delete and forget

This action is used for GDPR compliance, when a user requests that their personal data be erased.

WARNING: Deleting a viewer will delete all related data and permissions. This CANNOT be undone!
  1. In the left navigation, click Delete and forget.
  2. In the Delete and forget dialog, enter the word Delete.
  3. Click Delete and forget.
