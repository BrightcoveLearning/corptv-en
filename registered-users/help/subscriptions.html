Subscriptions

Registered user subscriptions include plan purchases and changes.

Subscription details

  • Plan name - Subscription plan name; click a plan name to see details
  • Store name - App store name
  • Status - Current subscription status
  • Start - Start date for the associated subscription
  • Trial Ends - End date for the user's trial period
  • Ends - End date for the associated subscription
  • Canceled - Date that the subscription was canceled
  • Show - Filter displayed content
Delete and forget

This action is used for GDPR compliance, when a user requests that their personal data be erased.

WARNING: Deleting a viewer will delete all related data and permissions. This CANNOT be undone!
  1. In the left navigation, click Delete and forget.
  2. In the Delete and forget dialog, enter the word Delete.
  3. Click Delete and forget.
