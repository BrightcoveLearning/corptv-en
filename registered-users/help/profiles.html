User Profiles

Each user gets one main profile, which cannot be deleted. They can add additional profiles to their account.

Profile details

  • Profile - The display name for the profile
  • Created - The creation date for the profile

  • Remove Profile - Click associated with a profile to delete it; activity and data related to this profile will no longer be tracked

    The main profile for an account cannot be removed.
Delete and forget

This action is used for GDPR compliance, when a user requests that their personal data be erased.

WARNING: Deleting a viewer will delete all related data and permissions. This CANNOT be undone!
  1. In the left navigation, click Delete and forget.
  2. In the Delete and forget dialog, enter the word Delete.
  3. Click Delete and forget.
