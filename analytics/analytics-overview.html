Viewing analytics data for {{ site.product_short }} content

In {{ site.product_short }} you have two options for viewing analytics data, those being:

  • Google Analytics
  • Video Cloud Analytics

Google Analytics

Google Analytics gathers data for all platforms with which you use {{ site.product_short }}. You can then use tools provided by Brightcove to view the data, as detailed in the Viewing Analytics Data document.

Video Cloud Analytics

At this time the following platforms support Video Cloud Analytics:

  • Web
  • iOS
  • Apple TV
  • Android
  • AndroidTV

For information on viewing Video Cloud Analytics see the documents in the Using the Analytics Module and Analytics Reports from the Analytics index page.
