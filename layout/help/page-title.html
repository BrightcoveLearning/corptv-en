Page name

Define the name of the page, which viewers will see in the Brightcove Beacon app.

  1. Enter the name for the page.
  • The name is used as part of the internal, system page ID.
  • The page name can be changed later.
  • Maximum characters allowed is beyond the practical limit you would want to use.
Page layout

Select a page layout. The layout will control the look and feel of the page and determines the number of playlists that can be added to a page.

  1. Choose one of the following layouts:
    • Playlist Grid
    • Hero (Autoplay Channel)
    • Hero (Featured Playlist)
    • Hero (Image)
    • Carousels
  2. Note that when a page layout is selected, the page preview will update to reflect the current selection.
  3. After making your selection, click Create that is positioned under the layout types.
