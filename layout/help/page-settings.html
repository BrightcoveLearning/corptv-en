From the page setting dialog you can:

  • Change the page title, and page title translations if using multiple languages.
  • Change page availability for schedule, devices and/or location. Click the corresponding toggle to alter values.
  • Archive the page. Note that no confirmation will be offered, the page is immediately archived.
