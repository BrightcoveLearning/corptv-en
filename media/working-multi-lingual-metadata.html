Working with Multi-Lingual Metadata

{{ page.description }}

To view and edit the details for a video, open the Media module, click All videos and click a video title link. The video details page will open.

Note: Before you can associate multi-lingual metadata with videos, you must configure a primary language and one or more additional languages as part of the Language Settings.

When the video details page opens, the video metadata will be displayed using the primary language. The language selector will display above the Video Information section and will display the primary language. Click the button to select another language.

Adding multi-lingual metadata

To add multi-lingual metadata to a video, select the language to add metadata for using the language selector.

The video properties will update and display the correct language version of the page.

To edit the video details, click the Edit button in a section or double-click in a section. Enter the localized value(s) and then click Save.

The following video details sections support multi-lingual metadata:

  • Video Information
  • Categorization
  • Text Tracks
  • Audio Tracks
  • Custom Fields
  • Images

Required fields

In order to save changes, all required fields for the primary language must have values. If required fields are missing for any of the additional languages, a message will be displayed but changes can be saved.
